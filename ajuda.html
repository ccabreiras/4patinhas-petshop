Ajuda e Documentação

Sobre o sistema

O 4 Patinhas Petshop é um projeto acadêmico desenvolvido para a disciplina de Fundamentos de Sistemas Web do curso de Análise e Desenvolvimento de Sistemas da PUC.

Na Fase 1, o site foi estruturado em HTML5 semântico. Na Fase 2, foram adicionados CSS personalizado, Bootstrap 5, JavaScript e formulários interativos para cadastro e agendamento.

Navegação

o site possui oito páginas:

  • Home (index.html) — página inicial com carrossel e visão geral das categorias;
  • Cães (caes.html) — produtos para cães;
  • Gatos (gatos.html) — produtos para gatos;
  • Peixes (peixes.html) — produtos para peixes;
  • Serviços (servicos.html) — serviços oferecidos pelo petshop;
  • Cadastro (cadastro.html) — formulário de cadastro de tutor e pet;
  • Agendamento (agendamento.html) — formulário de agendamento de serviço;
  • Ajuda (ajuda.html) — esta página de documentação.

Funcionalidades implementadas na Fase 2

CSS e Bootstrap

o site utiliza Bootstrap 5 para layout responsivo com navbar colapsável, carrossel de imagens na página inicial, cards para produtos e serviços, e formulários com validação visual. os estilos personalizados estão em css/style.css e utilizam variáveis CSS para as cores da marca.

JavaScript

o arquivo js/main.js implementa as seguintes funções:

  • relógio em tempo real: exibe data e hora no cabeçalho usando setInterval com atualização a cada segundo (função temporal);
  • data mínima: define automaticamente a data de hoje como limite mínimo no campo de data do agendamento;
  • tele-busca condicional: exibe ou oculta o campo de endereço conforme o método de entrega selecionado;
  • feedback de formulário: intercepta o envio, valida os campos e exibe mensagem de confirmação.

Formulário de cadastro

a página cadastro.html permite registrar o tutor (nome, CPF, data de nascimento, sexo via radio button, endereço, telefone e e-mail) e o pet (nome, espécie via select, raça, idade via number, porte via radio button e observações via textarea). campos obrigatórios utilizam o atributo required e validação visual Bootstrap.

Formulário de agendamento

a página agendamento.html permite agendar serviços de banho e/ou tosa. o tutor informa o serviço desejado (Banho, Tosa ou Banho e Tosa), o método de entrega (tele-busca ou presencial), a data e o horário. caso tele-busca seja selecionada, um campo de endereço é exibido automaticamente pelo JavaScript.

Acessibilidade

o site implementa os seguintes recursos de acessibilidade:

  • lang="pt-BR" em todas as páginas para leitores de tela;
  • atributos alt descritivos em todas as imagens;
  • skip link ("Pular para o conteúdo principal") para usuários de teclado e leitores de tela;
  • aria-label na navegação, no carrossel e no relógio;
  • aria-current="page" no link ativo da navegação em cada página;
  • aria-live="polite" no relógio para anunciar atualizações aos leitores de tela;
  • labels associados a todos os campos de formulário via atributos for e id.
